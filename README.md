# Pupfish Plugins

Claude Code plugin marketplace by Pupfish, LLC.

## Installation

Add this marketplace in Claude Code:

```
/plugin marketplace add Pupfish-LLC/claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| qlik-dev | Multi-agent development pipeline for Qlik Sense |

Install a plugin:

```
/plugin install qlik-dev@pupfish-plugins
```
